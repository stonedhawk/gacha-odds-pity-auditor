# Test Case Input: Standard Character Banner

Please audit the following gacha banner configuration:

## Currency & Cost
*   **Exchange Rate:** 100 Gems = $1 USD
*   **Cost Per Pull:** 150 Gems

## Rates & Probabilities
*   **Target Item:** "SSR Rank Character"
*   **Base Rate:** 0.6% chance to pull an SSR on any given non-pity pull.

## Pity Mechanics
*   **Soft Pity:** Starting at pull 70, the probability of pulling an SSR increases by a flat 2.0% per subsequent pull (e.g., Pull 70 = 2.6%, Pull 71 = 4.6%, etc.).
*   **Hard Pity:** An SSR is 100% guaranteed on pull 90 if one has not been obtained yet.

## The 50/50 Rule
*   When an SSR is pulled (either naturally or via pity), there is only a 50% chance it is the **Featured** banner character.
*   If the player loses the 50/50 (pulls a standard SSR), the next SSR pulled is 100% guaranteed to be the Featured character.
